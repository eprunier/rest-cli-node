
Usage: tsu amqp [<options>] <message>

Options:
    -e, --exchange <name>
    	Name of the exchange to send the message.

    -f, --file <file>
    	Load message from file.

    -h, --host <hostname>
    	AMQP server host.

    -k, --routing-key <routing key>
    	Routing key to use.

    -p, --port <port>
    	AMQP server port.

    -v, --verbose
    	Set verbose mode.
