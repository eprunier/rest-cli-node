
Usage: tsu amqp [<options>] <message>

Options:
    -e, --exchange <name>
    	Name of the exchange to send the message.

    -h, --host <hostname>
    	AMQP server host

    -k, --routing-key <routing key>
    	Routing key to use.

    -p, --port <port>
    	AMQP server port
 
    --rpc
	Set RPC mode. Send a message and wait for response.
